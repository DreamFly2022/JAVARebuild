# mysql 半同步复制，三个节点

[toc]

## 一、docker上启动三个mysql服务

```
sudo docker container run -p 3316:3306 --name mysql-1 -e MYSQL_ROOT_PASSWORD=root -d mysql:5.7
sudo docker container run -p 3326:3306 --name mysql-2 -e MYSQL_ROOT_PASSWORD=root -d mysql:5.7
sudo docker container run -p 3336:3306 --name mysql-3 -e MYSQL_ROOT_PASSWORD=root -d mysql:5.7
```

## 二、进入服务进行配置

```
sudo docker exec -it mysql-1 bash
sudo docker exec -it mysql-2 bash
sudo docker exec -it mysql-3 bash
apt-get update
apt-get install -y vim
```

### mysql-1 的配置

```
[mysqld]
pid-file	= /var/run/mysqld/mysqld.pid
socket		= /var/run/mysqld/mysqld.sock
datadir		= /var/lib/mysql

server_id = 16

sql_mode = NO_ENGINE_SUBSTITUTION,STRICT_TRANS_TABLES
log_bin = mysql-bin
binlog-format = Row
# 开启半同步复制
plugin_load_add='semisync_master.so'
rpl-semi-sync-master-enabled = 1


# GTID v.2 add
gtid_mode = ON
enforce-gtid-consistency
log-bin
log-slave-updates = 1

symbolic-links=0
```

## 三、配置

### 3.1 登陆主节点mysql-1：

```
mysql -h127.0.0.1 -P3316 -uroot -p

-- 创建复制用户
CREATE USER 'repl'@'%' IDENTIFIED BY '123456';

-- 设置权限
GRANT REPLICATION SLAVE ON *.* TO 'repl'@'%';

flush privileges;

mysql> show global variables like '%semi%';

| Variable_name                             | Value      |
+-------------------------------------------+------------+
| rpl_semi_sync_master_enabled              | ON         |
| rpl_semi_sync_master_timeout              | 10000      |

| rpl_semi_sync_master_wait_for_slave_count | 1          |
| rpl_semi_sync_master_wait_no_slave        | ON         |
| rpl_semi_sync_master_wait_point           | AFTER_SYNC |

### 3.2 登陆从节点mysql-2:

```
mysql -h127.0.0.1 -P3326 -uroot -p

-- 查看从节点状态
mysql> show slave status\G
Empty set (0.01 sec)

mysql> show global variables like '%semi%';
+---------------------------------+-------+
| Variable_name                   | Value |
+---------------------------------+-------+
| rpl_semi_sync_slave_enabled     | ON    |
| rpl_semi_sync_slave_trace_level | 32    |
+---------------------------------+-------+
2 rows in set (0.01 sec)



-- 设置同步
---  注意：使用 127.0.0.1 会导致连接不上
CHANGE MASTER TO
    MASTER_HOST='192.168.0.101',  
    MASTER_PORT = 3316,
    MASTER_USER='repl',      
    MASTER_PASSWORD='123456',   
    MASTER_LOG_FILE='mysql-bin.000001',
    MASTER_LOG_POS=739;
    
mysql> CHANGE MASTER TO
    ->     MASTER_HOST='192.168.0.101',
    ->     MASTER_PORT = 3316,
    ->     MASTER_USER='repl',
    ->     MASTER_PASSWORD='123456',
    ->     MASTER_LOG_FILE='mysql-bin.000001',
    ->     MASTER_LOG_POS=739;
Query OK, 0 rows affected, 1 warning (0.03 sec)

show slave status\G

## 启动复制，这一步骤很重要

show slave status\G
